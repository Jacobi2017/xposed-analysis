# xposed-analysis

## goal
在现有安卓权限选项（允许/禁止）的基础上，增加‘**伪装**’权限。对于伪装这个权限，用户实际使用时，类似于允许，区别就是实际访问到的该权限数据并不是真实的，而是一些虚假信息。以此达到为了能够使用某些应用程序，而不让其拥有某些其申请的权限的目的。
